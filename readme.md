## Install

```bash
$ npm install --save-dev eslint-config-fresh-react
```

## Usage

In `.eslintrc`:

```json
{
  "extends": [
    "eslint-config-fresh-react",
    "eslint-config-fresh-react/jsx-a11y"
  ]
}
```

In `package.json`:

```json
{
  "eslintConfig": {
    "extends": [
      "eslint-config-fresh-react",
      "eslint-config-fresh-react/jsx-a11y"
    ]
  }
}
```
